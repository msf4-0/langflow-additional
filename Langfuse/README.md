### ðŸ“¦ <b>Integration</b>
<b>Open the env where you have installed Langflow and pass on the following commands</b>
```shell

```
```shell
set LANGFLOW_LANGFUSE_PUBLIC_KEY=<your public key>
```
```shell
set LANGFLOW_LANGFUSE_HOST=http://localhost:3000
```
