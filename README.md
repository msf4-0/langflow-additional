### <b>Locally</b>
You can install Langflow from pip:

```shell
pip install langflow
```

Next, run:

```shell
python -m langflow
```
or
```shell
langflow # or langflow --help
```

## ðŸ“¦ Docker based Installation
### <b>Locally</b>
You can install Langflow Using Docker Desktop:

```shell
cd langflow-dev
```
Next, run:

```shell
docker build -f dev.Dockerfile . -t dev.dockerfile
```
Then, run:
```shell
docker-compose up -d
```
